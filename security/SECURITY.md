# Security Policy

## Supported Versions

This project is actively maintained. Security updates will be provided for the latest version.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

We take all security vulnerabilities seriously. Thank you for helping us keep this project safe.

### How to Report a Vulnerability

To report a security vulnerability, please send an email to **marie.challet@formasup-auvergne.fr** with the subject line: `[SECURITY] Vulnerability in formasup-bi-platform`.

Please include the following information in your report:

- A detailed description of the vulnerability, including the steps to reproduce it.
- The version of the project you are using.
- Any screenshots, logs, or scripts that are relevant to the vulnerability.

### What to Expect

- You will receive an acknowledgment of your report within 48 hours.
- We will investigate the vulnerability and keep you informed of our progress.
- We will notify you when the vulnerability has been patched.
- We will credit you for your discovery, unless you prefer to remain anonymous.

### Security Best Practices

- **Do not** disclose the vulnerability publicly until it has been patched.
- **Do not** attempt to access or modify data that does not belong to you.
- **Do** provide as much information as possible to help us resolve the issue quickly.
